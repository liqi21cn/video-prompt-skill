# Micro-Expression Library (v3.2)

> 50 scene-tagged micro-expression presets organized by emotion category. Same content as §9.2 of `full-prompt-v3.2.md`, extracted for on-demand loading.

## Formula

Each entry follows the same five-element pattern:

> **eye landing (眼神落点) + brow change (眉眼变化) + mouth change (嘴角变化) + breath pause (呼吸停顿) + small action (小动作) + applicable scene (适用场景)**

## How to use

1. **Match emotion → category** — Map `originalText` emotional context to the closest category.
2. **Match scene → entry** — Pick the entry whose 适用场景 best fits.
3. **Copy the physical description verbatim** into the "③ micro-expression" dimension of the shot in `sceneDesc`. Don't include `#NN` or the scene tag.
4. **Trim if needed** — For short shots, keep just the 1-2 most distinctive details (eye landing + one small action is often enough).
5. **Stack if needed** — Complex emotion can combine two entries (e.g. #18 悔意涌上 + #01 强忍委屈).
6. **For English platforms** — Translate the physical observations (which translate cleanly), not the scene tag names.
